What is a Web Browser?

A Web Browser is computer software that allows individuals to visually interact with the internet.

The Internet by itself is a superhighway of information built on network infrastructure that connects computers and devices across the world, and web browser technology allows individuals to freely navigate this infrastructure by typing a destination in the web address bar, which would then take them to a “website” (Mozilla, 2021).

"The Web does not just connect machines, it connects people."

-Sir Tim Berners-Lee

The Inception of Web Browsers

In the 1960s and due to the cold war’s rising tensions, world governments funded many scientific and technical ventures.

The earliest form of Internet was known as ARPANet, which was created in 1962 by a scientist named J.C.R Licklider for the Advanced Research Project Agency (ARPA).

However, this version of the Internet was mostly for Governments and researchers to exchange information through databases (History.com Editors, 2024). It was not until the creation of the WorldWideWeb (later named “Nexus”), by Sir Tim Berners-Lee, that individuals could navigate the Internet more easily through a “Web Browser”.
